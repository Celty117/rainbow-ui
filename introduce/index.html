# Rainbow-UI

Rainbow-UI 是一个好用的 UI 框架，提供了一些常用组件，适合 PC 端和移动端使用。

组件：按钮、输入框、网格、布局、Toast、Tabs

# 目的

这是我在学习 Vue 的过程中尝试写的一个 UI 框架（造的轮子），希望对你有用。
